
## Reference

### 多语言配置

* localstorage 配置项如下
```js
localStorage.setItem('grpLocale', 'en-US') 
localStorage.setItem('grpLocale', 'zh-CN')  
```

* 目前仅支持中英文设置，默认显示英文。
